# Практическое задание: Thread vs Runnable/Callable

## Описание тестов

### testRunnableTask
Проверяет создание и выполнение задачи через `Runnable`. `CounterTask` должен увеличить счетчик на 1.

### testCallableTask
Проверяет создание и выполнение задачи через `Callable`. `SumTask` должен суммировать числа от 1 до n и вернуть результат.

### testMultipleThreads
Проверяет работу нескольких потоков с одной задачей. Все потоки должны выполнить `CounterTask` и увеличить счетчик.

### testThreadInterruption
Проверяет корректную обработку прерывания потока. Поток должен проверить флаг прерывания и корректно завершиться.

### testCallableWithException
Проверяет обработку исключений в `Callable`. Исключения должны быть обернуты в `ExecutionException`.

## Классы для реализации

- **CounterTask**: Задача, реализующая Runnable, которая увеличивает счетчик
- **SumTask**: Задача, реализующая Callable<Integer>, которая суммирует числа от 1 до n
